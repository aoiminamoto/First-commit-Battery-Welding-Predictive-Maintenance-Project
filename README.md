# 🔋 Battery Welding Predictive Maintenance

### Project Overview
This project aims to analyze *simulated* welding failure (NG) data sourced from an Andon system. Utilizing machine learning algorithms, we identified the key sensor parameters responsible for triggering these failures.

### Key Highlights
* **Data Scale:** Processed a large-scale dataset—comprising non-production simulation data—spanning multiple production shops (e.g., Coating, Module).
* **Model Selection:** Evaluated various algorithms and ultimately selected **Gradient Boosting** as the Champion Model.
* **Performance Metrics:** Achieved a significant improvement in the **MCC Score** when handling highly imbalanced data.
